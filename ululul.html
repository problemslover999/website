Unordered List

An unordered list is a type of list in HTML that is used to display items in no particular order. It is created using the <ul> tag, and each item within the list is defined using the <li> tag. Unordered lists are typically rendered with bullet points or other markers to indicate each item, making them useful for presenting information that does not require a specific sequence.


Example of an unordered list:

  • Item 1
  • Item 2
  • Item 3
